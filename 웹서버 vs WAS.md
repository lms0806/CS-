### Web Server VS Web Application Server

정적 페이지 vs 동적 페이지

정적 페이지(Static Pages)
 - 바뀌지 않는 페이지
 - 웹 서버는 파일 경로 이름을 받고, 경로와 일치하는 file contents를 반환
 - 항상 #동일한 페이지를 반환

동적 페이지(Dynamic Pages)
 - 인자에 따라 바뀌는 페이지
 - 웹 서버에 의해 실행되는 프로그램을 통해 만들어진 결과물
 - 개발자는 Sevlet에 doGet() 메소드를 구현함

웹 서버
 - 하드웨어와 소프트웨어로 구분됨
